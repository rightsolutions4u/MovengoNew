Tranship - Shipping / Logistics / Transportation Template

Tranship is a responsive HTML Template created for transportation, logistics, shipping companies and small freight business. This is powerful HTML template with full of customization options and features.

Tranship is based on Bootstrap 1170px grid system. All files are clearly organised we believe it will be easy to use and edit them. The design is very clean, elegant and modern. Any one can customize this template as his wishes

  • Version: 1.2
  • Created: 04/20/2017

;(function($){
  'use strict';
      // Slider
      var $slider = $('#slider');
      if ($slider.length > 0 ) {
        $slider.carousel({ interval:6000, pause: 'null' });
      }

      // Quote Carousel
      var $carousel = $('#carousel');
      if ($carousel.length > 0 ) {
        $carousel.carousel({ interval:4000, pause: 'hover' });
      }
      
      // Logo Carousel
      var $logo_carousel = $('.logo-carousel');
      if ($logo_carousel.length > 0 ) {
          $logo_carousel.owlCarousel({
            items: 3, loop: true, margin: 6, responsive:{ 600:{ items:4 }, 1000:{ items:6 } }
        });
      }

      // Parallax
      var $parallax = $('.has-parallax');
      if ($parallax.length > 0 ) {
        $parallax.parallaxie({ speed: 0.5, offset: 50 });
      }

      // Icon Rollover -Service
      var $icon_list_over =  $('.icon-list-over');
      if ($icon_list_over.length > 0) {
        $icon_list_over.children('li').on('mouseenter mouseleave', function() {
          $(this).find('.icon').toggleClass('hover');
        });
      }
      // Active page menu when click
      var url = window.location.href;
      var $nav_link = $(".nav li a");
      $nav_link.each(function() {
          if (url === (this.href)) {
              $(this).closest("li").addClass("active");
          }
      });
})(jQuery);

How to Set Email Address for Form Since version 1.1.1

First open the php file located into form folder. Files are 'contact.php', 'quote-request.php', 'subscribe.php'. On each file you will saw bellow code. Just update your sitename, your email and name. If you want to add multiple email then just duplicate the line '$emailTO[] = array...' and add another email

// Enter Your Sitename
$sitename = 'your site';

// Enter your email addresses: @required
$emailTO[] = array( 'email' => 'your mail', 'name' => 'your name' );

If you want to add BCC / CC email address then just enable below line and update same way as email too.

//$emailBCC[] = array( 'email' => 'email@yoursite.com', 'name' => 'Your Name' );

//$emailCC[] = array( 'email' => 'email@yoursite.com', 'name' => 'Your Name' );
